## 1. BASI Comprehensive 600 小时体系拆解

**总时长：600 小时（BASI 官方要求）**

| 分类 | 小时数 | 内容说明 |
|------|--------|-----------|
| 12 个模块 | 84h | 理论、解剖、动作 repertoire（线下） |
| Revision（复盘课） | 76h | 4×4h + 多次复盘课程（必须） |
| Classes Pro | 30 节（约 45h） | cueing / 教学方法练习 |
| Observation | 100h | 观察 Mat + Machines 课程 |
| Personal Practice | 100h | 自主练习（Mat / Machines） |
| Supervised Practice | 40h | 教师监督练习 |
| Teaching Practice | 200h | 教学练习（mock teaching） |

完成全部小时数 + 考试后，可获得 BASI 国际认证（全球认可）。

## 3. 完整转型路线图（18–24 个月）

### Phase 0（当前）｜基础建立（0–2 个月）
- 持续上 Rituel 课程（每周 3–4 节）  
- 建立呼吸 / 核心 / 稳定基础  
- 开始整理老师口令  
- 开始解剖基础学习  

### Phase 1｜Prerequisite 70 小时（2–4 个月）
- 累积 50h Mat  
- 累积 20h Machines  
- 向 Rituel 预约报名面试  
- 撰写动机信  

### Phase 2｜进入 BASI 课程（4–6 个月）
- 参加 12 Modules  
- 每月 1 个模块（周末）  
- 每次模块后参加 Revision  
- 每周至少 1 次 Class Pro  
- 开始 Observation（累计 20–30h）

### Phase 3｜Teaching Practice（7–15 个月）
- 完成 100h Observation  
- 完成 100h Personal Practice  
- 完成 40h Supervised Practice  
- 完成 200h Teaching Practice  
- 撰写 mid-term case study  

### Phase 4｜Final Exam 准备（16–20 个月）
- 完成 final case study  
- 完成 teaching practical  
- 完成理论考试  
- 检查小时数是否全部达标  

### Phase 5｜成为正式 BASI 教练（20–24 个月）
- 拿到 BASI 国际证书  
- 申请 CQP + 法国教练职业卡（如在法国教学）  
- 可在巴黎精品工作室兼职教学（Rituel / Keana / A-Lyne）
