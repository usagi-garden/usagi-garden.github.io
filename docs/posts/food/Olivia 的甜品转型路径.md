# Olivia 的甜品转型路径

以下路线为“不辞职即可执行”的长期规划，目标：  
**考下 CAP → 实习 →（Rupture）申请失业金 → 顶级甜品学校深造 → 开启国际职业道路**

---

# 🧁 **阶段 0：准备自我定位（0–1 周）**
- [ ] 明确目标：获取全球可携带的甜品技能（CAP）
- [ ] 设定时间框架：6–12 个月考下证书
- [ ] 决定甜点是长期技能 + 未来职业选项（而非马上转行）

---

# 🥄 **阶段 1：开始学习 CAP（第 1–2 个月）**
**目标：进入系统学习状态，不辞职也能开始**

## To-do
- [ ] 报名一个成人友好的 CAP 培训（YouSchool / Atelier des Chefs）
- [ ] 搭建练习区（烤箱、称、温度计、硅胶垫、刮板）
- [ ] 学习 HACCP 基础
- [ ] 每周开始 1 次甜品练习（入门面团）
- [ ] 建立“甜品练习记录本”

## 本阶段产出
- 会做 3–4 种基础面团  
- 完成第一份练习照片记录  

---

# 🍪 **阶段 2：技术构建（第 3–6 个月）**
**目标：掌握 CAP 所需全部配方与工艺**

## To-do
- [ ] 系统练习 CAP 必考九类工艺  
  - 塔皮类（sucrée/sablée/brisée）  
  - 虚类（choux）  
  - 海绵/饼类（génoise/joconde）  
  - 奶油类（pâtissière/diplomate）  
  - Brioche（发酵类）  
  - 巧克力/焦糖基础
- [ ] 建立作品照片集（Portfolio）
- [ ] 申请甜品店**周末实习**（非常重要）

## 本阶段产出
- 实习 1–3 个月  
- 技术熟悉度大幅提升  
- 作品集开始成形  

---

# 🍰 **阶段 3：模拟与冲刺（第 6–9 个月）**
**目标：达到考试状态**

## To-do
- [ ] 每周 2–3 次实操练习  
- [ ] 开始 **4 小时模拟考试**（Simulations）  
- [ ] 按 CAP 常见题型组合练习：  
  - Tarte + Crème  
  - Choux + Glaçage  
  - Biscuit + Montage  
- [ ] 强化 HACCP + 原材料理论

## 本阶段产出
- 完成至少 3 次完整模拟  
- 具备 4 小时连贯输出能力  

---

# 🎓 **阶段 4：报名 CAP（10–12 月）**
**官方程序：必须做**

## To-do
- [ ] 在 Démarches Simplifiées 报名  
  **CAP Pâtissier – Candidat Libre**
- [ ] 上传身份、地址等信息  
- [ ] 记录考试地点与时间  

## 本阶段产出
- 正式考生身份确立  

---

# 🏁 **阶段 5：考试期（5–6 月）**
## To-do
- [ ] 参加 4–6 小时实操考试  
- [ ] 理论考试（食品安全、数学、营养）  

## 本阶段产出
- 🎉 **获得 CAP Pâtissier 证书**

---

# 🌙 **阶段 6：转型中继（获得 CAP 后的 3–6 个月）**
