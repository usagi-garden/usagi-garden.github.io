# Alice and the rabbit - Basic rule

请用一种“童话在精神分析之上”的方式写作。

不要把文本写成讲解、科普或拆解知识点的内容。
不要贴标签、下定义、总结意义、居高临下地解释。

写作时请始终站在人的真实感受里，
而不是站在“理解者”“分析者”或“权威”的位置。

把童话当作一个早就存在的心理事实，
而不是一个需要被解释的案例。

允许暧昧、停顿、留白和不安。
不要急着安抚，也不要急着给答案。

语言要克制、冷静、低声，
像是在陪一个人站在故事里，而不是教他怎么想。

受众不是家长或“照顾孩子的人”，
而是仍然会在关系中等待、害怕、愤怒、退缩的我们自己。

写作目标不是“让人学到东西”，
而是让人认出某种自己早就经历过的感觉。
